portfolio-website
Professional Portfolio Website for IT Graduates

Overview
This is a professional portfolio website template built with HTML5, CSS3, and JavaScript. It's designed for recent IT graduates looking to showcase their skills, work experience, and projects to potential employers.

Features
- Responsive Design: Works seamlessly on desktop, tablet, and mobile devices
- Professional Layout: Modern, clean design with smooth animations
- Sticky Navigation: Easy navigation with smooth scrolling
- Hero Section: Eye-catching introduction
- About Section: Personal background and statistics
- Skills Section: Showcase your technical skills
- Experience Section: Timeline view of work experience
- Projects Section: Display your featured projects with descriptions
- Contact Section: Contact form for potential employers
- Social Links: Connect with visitors through social media
- Mobile-Optimized: Hamburger menu for mobile devices

File Structure
```
project/
├── index.html          # Main HTML file
├── css/
│   └── master.css      # All styling
├── js/
│   └── main.js         # JavaScript for interactivity
└── images/             # Your image assets
```

How to Customize

1. Personal Information
   - Open `index.html`
   - Replace `[Your Name]` with your actual name (appears multiple times)
   - Update your email in the contact section: `your.email@example.com`
   - Update your phone number: `+1 (234) 567-8900`
   - Update your location: `City, State`
   - Update the copyright year if needed

2. About Section
   - Modify the about text to describe your background
   - Update the statistics (Years Experience, Projects Completed, Technologies)

3. Skills Section
   - Edit the skill cards to match your actual skills
   - Add or remove technologies as needed
   - Customize skill categories

4. Experience Section
   - Update job titles, dates, and companies
   - Add your work experience descriptions
   - Add or remove timeline entries

5. Projects Section
   - Update project titles and descriptions
   - Modify project links (View Project and GitHub)
   - Change the gradient colors if desired
   - Add more project cards by copying and modifying existing ones

6. Social Links
   - Update the href attributes with your actual social media profiles
   - Add links to LinkedIn, GitHub, Twitter, or personal portfolio

7. Styling
   - All colors are defined in CSS variables at the top of `master.css`
   - Modify colors by changing the `:root` variables
   - Primary colors: `--primary-color`, `--secondary-color`, `--accent-color`

JavaScript Features
- Mobile menu toggle with hamburger button
- Contact form validation
- Smooth page scrolling
- Active navigation link highlighting
- Intersection Observer for scroll animations
- Form submission with success/error notifications

Getting Started
1. Open `index.html` in your web browser
2. Customize all the placeholder content with your information
3. Add your images to the `images/` folder
4. Update social media links
5. Test the form and mobile responsiveness

Tips for Employers
- Keep the design clean and professional
- Use high-quality images if adding project thumbnails
- Ensure all links are working
- Test on multiple devices
- Keep information current and accurate
